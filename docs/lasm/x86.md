# Instructions

## AAA – ASCII Adjust After Addition

Adjusts AL after adding two ASCII digits to form a valid BCD result.

```hex
37     ; aaa
```

## AAD – ASCII Adjust AX Before Division

Adjusts AX by converting two unpacked BCD digits in AH and AL into a binary number in AL, clearing AH before a division.

```hex
D5 0A         ; aad
D5 <imm8>     ; aad <imm8>
```

## AAM – ASCII Adjust AX After Multiply

Adjusts AX after multiplying two unpacked BCD digits to form unpacked BCD digits in AH and AL.

```hex
D4 0A         ; aam
D4 <imm8>     ; aam <imm8>
```

## AAS – ASCII Adjust AL After Subtraction

Adjusts AL after subtracting two ASCII digits to form a valid BCD result.

```hex
3F     ; aas
```
