# Instructions

## ASCII

### AAA - ASCII Adjust After Addition

Adjusts AL after adding two ASCII digits to form a valid BCD result.

Affected flags: AF, CF

```hex
37    ; aaa
```

### AAD - ASCII Adjust AX Before Division

Adjusts AX by converting two unpacked BCD digits in AH and AL into a binary number in AL, clearing AH before a division.

Affected flags: SF, ZF, PF

```hex
D5 0A        ; aad
D5 <imm8>    ; aad <imm8>
```

### AAM - ASCII Adjust AX After Multiply

Adjusts AX after multiplying two unpacked BCD digits to form unpacked BCD digits in AH and AL.

Affected flags: SF, ZF, PF

```hex
D4 0A        ; aam
D4 <imm8>    ; aam <imm8>
```

### AAS - ASCII Adjust AL After Subtraction

Adjusts AL after subtracting two ASCII digits to form a valid BCD result.

Affected flags: AF, CF

```hex
3F    ; aas
```

## ALU

### ADC - Add With Carry

Adds the destination operand, the source operand, and the carry flag and stores the result in the destination operand

Affected flags: OF, SF, ZF, AF, CF, PF

```hex
14 <imm8>                                 ; adc al, <imm8>
(66) (REX.W) 15 <imm16|32>                ; adc ax|eax|rax, <imm16|32>
80 <ModR/M /2> <imm8>                     ; adc r/m8, imm8
(66) (REX.W) 81 <ModR/M /2> <imm16|32>    ; adc r/m16|32|64, <imm16|32>
(66) (REX.W) 83 <ModR/M /2> <imm8>        ; adc r/m16|32|64, <imm8>
10 <ModR/M>                               ; adc r/m8, r8
(66) (REX.W) 11 <ModR/M>                  ; adc r/m16|32|64, r16|32|64
12 <ModR/M>                               ; adc r8, r/m8
(66) (REX.W) 13 <ModR/M>                  ; adc r16|32|64, r/m16|32|64
```

### ADCX - Unsigned Integer Addition of Two Operands With Carry Flag

Adds the destination operand, the source operand, and the carry flag and stores the result in the destination operand

Affected flags: CF

```hex
66 (REX.W) 0F 38 F6 <ModR/M>    ; adcx r32|64, r/m32|64
```

### ADD - Add

Adds the destination operand and the source operand and stores the result in the destination operand

Affected flags: OF, SF, ZF, AF, CF, PF

```hex
04 <imm8>                                 ; add al, <imm8>
(66) (REX.W) 05 <imm16|32>                ; add ax|eax|rax, <imm16|32>
80 <ModR/M /0> <imm8>                     ; add r/m8, imm8
(66) (REX.W) 81 <ModR/M /0> <imm16|32>    ; add r/m16|32|64, <imm16|32>
(66) (REX.W) 83 <ModR/M /0> <imm8>        ; add r/m16|32|64, <imm8>
00 <ModR/M>                               ; add r/m8, r8
(66) (REX.W) 01 <ModR/M>                  ; add r/m16|32|64, r16|32|64
02 <ModR/M>                               ; add r8, r/m8
(66) (REX.W) 03 <ModR/M>                  ; add r16|32|64, r/m16|32|64
```
