# LHOHO

## Purpose

LHOHO is a simple, lightweight tool that displays the remaining time until Christmas or prints an ASCII Christmas tree.

## Usage

Run the tool from the command line:

```bash
lhoho <command> [options]
```

### Examples

```bash
lhoho tree
lhoho tree 15
lhoho countdown
```

## Commands

- `tree [height]`: Print an ASCII Christmas tree (default height: 10)

- `countdown`: Display the remaining time until Christmas

## Options / Flags

- `-h`, `--help`: Show help message

- `-v`, `--version`: Display the version
